# administración_de_pedidos_utilizando_estructuras_de_datos
Se incluye interfaz grafica
Al iniciar el programa deberá mostrarse el siguiente menú:

Seleccione la estructura a utilizar:
1. Pila
2. Cola
3. Lista

Después de elegir la estructura, deberá mostrarse el siguiente menú:

1. Agregar pedido
2. Atender pedido
3. Mostrar pedidos
4. Salir



Dato de cada pedido

• Producto solicitado

Funcionamiento

Si seleccionas PILA:
Los pedidos se atenderán del último registrado al primero.

Si seleccionas COLA:
Los pedidos se atenderán en el orden en que llegaron.

Si seleccionas LISTA:
Los pedidos se almacenarán en una lista y podrás eliminar un pedido indicando su posición.

Entregables

1. Programa en Python
- Menú principal
- Uso de ciclos
- Uso de condiciones
- Uso de listas
- Agregar pedidos
- Atender/eliminar pedidos
- Mostrar pedidos
- Salir
